
## API

```javascript
function useCreation<T>(factory: () => T, deps: any[]): T;
```

### 参数

| 参数    | 说明                                         | 类型                   | 默认值 |
|---------|----------------------------------------------|------------------------|--------|
| factory | 用来创建所需对象的函数  | () => any | -      |
| deps | 传入依赖变化的对象  | any[] | -      |
