## API
| 属性名称  | 属性说明       | 类型   | 默认值 | 是否必须 |
| :-------- | :------------- | :----- | :----- | :------- |
| type      | icon 类型标识  | string | -      | 是       |
| className | 额外样式       | string | -      | 否       |
| style     | 内联样式       | obj    | -      | 否       |
| ...       | svg 支持的属性 | -      | -      | 否       |
