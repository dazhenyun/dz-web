## API

| 参数        | 说明                                    | 类型   | 默认值 | 版本 |
| ----------- | --------------------------------------- | ------ | ------ | ---- |
| name        | 参数名,必填                             | string | -      |      |
| listFormSet | 每行的输入框配置，参考 GForm 的 formSet | array  | -      |      |
| hasAdd      | 是否显示新增按钮                        | bool   | true   |      |
| hasDel      | 是否显示删除按钮                        | bool   | true   |      |
| hasHead     | 是否有头部，呈现表格形式                | bool   | false  |      |
